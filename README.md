## 🧭 Overview

StockGyan AI helps users make better-informed decisions by combining **machine-learning price prediction** with **real-time market analytics** in a clean, responsive interface. Instead of static charts, it surfaces multi-horizon forecasts and interactive visualizations so trends, risk, and momentum are easy to read at a glance.

> ⚠️ **Disclaimer:** StockGyan AI is built for educational and analytical purposes only. It is **not** financial advice. Always do your own research before making investment decisions.

---

## ✨ Key Features

- 📊 **Real-time market insights** — live data surfaced through an intuitive dashboard
- 🔮 **Multi-horizon predictions** — short-, mid-, and long-term price forecasts from ML models
- 📈 **Interactive charts** — zoomable, hover-rich visualizations for price and trend analysis
- ⚡ **Responsive UI** — fast, mobile-friendly experience built with React + Tailwind
- 🧮 **Data-driven analytics** — risk and trend signals derived from historical market data

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/sunildhakad01/StockGyan--Stock-price-prediction-using-ML.git

# Navigate into the project
cd StockGyan--Stock-price-prediction-using-ML

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.
